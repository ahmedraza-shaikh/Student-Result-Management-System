# 🎓 Student Result Management System

A web-based Student Result Management System built with
Python, Flask, SQLite, HTML & Bootstrap.

## 🚀 Features
- ➕ Add students with subject marks
- 📊 Auto calculate percentage
- 🏅 Auto assign grades (A+/A/B/C/D/F)
- ✅ Auto Pass/Fail status
- 🏆 Top 3 Toppers leaderboard
- 🔍 Search by name or roll number
- ✏️ Edit student records
- 🗑️ Delete with confirmation
- 🔴 Red highlight for failed students

## 🛠️ Tech Stack
| Layer    | Technology        |
|----------|-------------------|
| Backend  | Python + Flask    |
| Database | SQLite            |
| Frontend | HTML + Bootstrap  |
| Language | Python 3.x        |

## ⚙️ How to Run

1. Clone the repository
   git clone https://github.com/ahmedraza-shaikh/student-result-management-system.git

2. Install Flask
   pip install flask

3. Run the app
   py app.py

4. Open browser and go to
   http://127.0.0.1:5000
